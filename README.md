## ✅ 스터디 방법

1. 이 저장소를 `fork` 한다.
2. 생성된 원격 저장소에 `이름`으로 폴더를 생성한다.
3. 생성된 폴더에 알고리즘 `스터디 날짜`별 폴더를 만든다.
4. 자신의 `소스코드`를 업로드 한다.
5. 원본 저장소로 `Pull Request`를 한다.
6. 다른 사람들의 PR을 보고 자유롭게 코드리뷰를 한다.
7. 폴더 구조 예시) `홍길동/0204`, `황태희/0107`

<span style="color:yellow">⚠️***주의! 다른 사람의 폴더를 건드리면 안됩니다!***</span>
